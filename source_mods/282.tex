\section[Problem \#282: The Ackermann function]{The Ackermann function}
\label{sec:problem_282}

For non-negative integers $m$, $n$, the Ackermann function $A(m,n)$ is defined as follows:
$$
A(m,n) = \begin{cases}
n+1 & $\text{ if }m = 0$\cr
A(m-1,1) & $\text{ if }m > 0 \text{ and } n = 0$\cr
A(m-1,A(m,n-1)) & $\text{ if }m > 0 \text{ and } n > 0$\cr
\end{cases}$$

For example $A(1,0) = 2$, $A(2,2) = 7$ and $A(3,4) = 125$.

Find $\displaystyle\sum_{n=0}^6 A(n,n)$ and give your answer mod $14^8$.
